# Harsh Soni — Portfolio & CV Website 🚀

A dynamic, responsive, and database-driven personal portfolio website built using **Django** and **Python**. This project serves as a digital resume showcasing my skills, projects, education, and certifications as an AI Engineer and Backend Developer.

![Theme](https://img.shields.io/badge/Theme-White%20%26%20Blue-2563eb?style=flat-square)
![Framework](https://img.shields.io/badge/Framework-Django-092E20?style=flat-square&logo=django)
![Language](https://img.shields.io/badge/Language-Python-3776AB?style=flat-square&logo=python)

## ✨ Features
* **Dynamic Database Integration:** Work experience and projects can be easily updated via the built-in Django Admin panel without touching the HTML.
* **Modern UI/UX:** Clean white and blue color scheme with a custom interactive cursor, smooth scroll animations, and CSS noise overlays.
* **Fully Responsive:** Optimized for both desktop and mobile viewing using a combination of custom CSS and Bootstrap 5.
* **Animated Elements:** Includes an infinite-scroll tech stack ticker and numerical stat counters.

## 🛠️ Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML5, Custom CSS3, Bootstrap 5
* **Database:** SQLite (default Django DB)
* **Fonts:** Google Fonts (Outfit, Playfair Display, DM Mono)
